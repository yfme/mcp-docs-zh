## 图片

### 使用 Markdown

[Markdown 语法](https://www.markdownguide.org/basic-syntax/#images)允许您使用以下代码添加图片

```md
![标题](/path/image.jpg)
```

注意，图片文件大小必须小于 5MB。否则，我们建议将图片托管在 [Cloudinary](https://cloudinary.com/) 或 [S3](https://aws.amazon.com/s3/) 等服务上。然后您可以使用该 URL 进行嵌入。

### 使用嵌入

要获得更多的图片自定义选项，您也可以使用[嵌入](/writing-content/embed)来添加图片

```html
<img height="200" src="/path/image.jpg" />
```

Mintlify 支持在 Markdown 中使用 [HTML 标签](https://www.markdownguide.org/basic-syntax/#html)。如果您更喜欢使用 HTML 标签而不是 Markdown 语法，或者需要创建具有无限灵活性的文档，这将很有帮助。

</Tip>

### iFrame

在文档中加载另一个 HTML 页面。最常用于嵌入视频。

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
