## 基础用法

### 内联代码

要将`单词`或`短语`表示为代码，请将其用反引号(`)括起来。
```
要将`单词`或`短语`表示为代码，请将其用反引号(`)括起来。
```

### 代码块

使用[围栏式代码块](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks)，将代码用三个反引号括起来，并在开头的反引号后面标注代码片段的编程语言以获取语法高亮。您也可以选择性地在编程语言后面写入代码的名称。

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
