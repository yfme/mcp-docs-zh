## 标题
最适合用作章节标题。
```md
## 标题
```

### 副标题
最适合用作小节标题。
```md
### 副标题
```

<Tip>
每个**标题**和**副标题**都会创建一个锚点，并显示在右侧的目录中。
</Tip>

## 文本格式
我们支持大多数 Markdown 格式。只需在文本周围添加 `**`、`_` 或 `~` 即可设置格式。

| 样式 | 如何书写 | 效果 |
| ------------- | ----------------- | --------------- |
| 粗体 | `**粗体**` | **粗体** |
| 斜体 | `_斜体_` | *斜体* |
| 删除线 | `~删除线~` | ~删除线~ |

您可以组合使用这些格式。例如，写入 `**_粗体和斜体_**` 得到 **_粗体和斜体_** 文本。

您需要使用 HTML 来写上标和下标文本。也就是说，在文本周围添加 `<sup>` 或 `<sub>`。

| 文本大小 | 如何书写 | 效果 |
| ----------- | ------------------------ | ---------------------- |
| 上标 | `<sup>上标</sup>` | <sup>上标</sup> |
| 下标 | `<sub>下标</sub>` | <sub>下标</sub> |

## 链接到页面
您可以通过将文本包装在 `[]()` 中来添加链接。写入 `[链接到谷歌](https://google.com)` 得到 [链接到谷歌](https://google.com)。

文档中的页面链接需要是根相对的。基本上，您应该包含完整的文件夹路径。例如，`[链接到文本](/writing-content/text)` 链接到我们组件部分的"文本"页面。

相对链接如 `[链接到文本](../text)` 打开速度会较慢，因为我们无法轻易优化它们。

## 引用块

### 单行
要创建引用块，在段落前面添加 `>`。

> 多萝西跟着她参观了城堡中许多美丽的房间。

```md
> 多萝西跟着她参观了城堡中许多美丽的房间。
```

### 多行
> 多萝西跟着她参观了城堡中许多美丽的房间。
>
> 女巫让她清洗锅碗瓢盆，打扫地板，并往火里添柴。

```md
> 多萝西跟着她参观了城堡中许多美丽的房间。
>
> 女巫让她清洗锅碗瓢盆，打扫地板，并往火里添柴。
```

### LaTeX

<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>

```md
<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>
```
