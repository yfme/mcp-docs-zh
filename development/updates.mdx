<Update label="2025-02-14" description="Java SDK 发布">
  - 我们很高兴地宣布，由 VMware Tanzu 的 Spring AI 开发的 Java SDK 现在成为 MCP 的官方 [Java SDK](https://github.com/modelcontextprotocol/java-sdk)。
    这与我们现有的 Kotlin SDK 一起加入了我们不断增长的支持语言列表。
    Spring AI 团队将维护该 SDK，作为模型上下文协议组织的重要组成部分。
    我们很高兴欢迎他们加入 MCP 社区！
</Update>

<Update label="2025-01-27" description="Python SDK 1.2.1">
  - MCP Python SDK 的 [1.2.1](https://github.com/modelcontextprotocol/python-sdk/releases/tag/v1.2.1) 版本已发布，
    提供了重要的稳定性改进和错误修复。
</Update>
<Update label="2025-01-18" description="SDK 和服务器改进">

</Update>

</Update>
